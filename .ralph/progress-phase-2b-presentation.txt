## Codebase Patterns (Phase 2B, presentation-format handlers)

- Pandoc helpers (ensure-pandoc.sh, render-pdf.sh, source-hash.sh) from Phase 2A are the substrate — inherit them, don't reinvent.
- Page-selection logic (tag / folder / file / `all` + deterministic sort) is duplicated across generate-book and generate-slides. If generate-mindmap or generate-infographic makes it a third copy, lift it into `.claude/skills/generate/lib/select-pages.sh`.
- Wikilink sed pass is the exact same one-liner in every handler — another candidate for the shared lib/.
- Marp CLI installs via `pnpm add -g` or `npm install -g`; fall back to `npx @marp-team/marp-cli` if install fails. No brew formula.
- Vault-local template overrides live at `<vault>/.templates/<type>/` — lets vaults ship house style without polluting the shared handler.
