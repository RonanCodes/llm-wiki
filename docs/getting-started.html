Personal Knowledge Engine

LLM Wiki

A compounding knowledge base built by LLMs. Claude Code writes your wiki. Obsidian is where you read it. Your knowledge grows with every source and every question.

Inspired by Karpathy's LLM Wiki pattern
llm-wiki
You (terminal) You (Obsidian) | | |-- /ingest <source> |-- Browse wiki pages |-- /query "question" |-- Graph view |-- /lint |-- Backlinks |-- /promote |-- Search | | '-- Claude Code --writes--> vaults/<name>/wiki/*.md
01

Quick Install

Three commands. The install script checks your environment and offers to install anything missing via Homebrew.

$ git clone https://github.com/RonanCodes/llm-wiki.git
$ cd llm-wiki
$ ./install.sh  # interactive — asks before installing each tool
02

What You Need

Core tools are required. Everything else installs automatically the first time you need it.

Claude Code required

The AI engine. Reads sources, writes wiki pages, answers questions, maintains cross-references.

npm i -g @anthropic-ai/claude-code

Documentation →

Obsidian required

Your wiki viewer. Browse pages, graph view, backlinks, full-text search.

brew install --cask obsidian

Download →

Git required

Version control. Each vault is its own git repo with full history.

brew install git

Node.js required

Runtime for Claude Code and npm packages.

brew install node

Download →

yt-dlp auto

YouTube transcript extraction. Installs automatically on first YouTube ingest.

brew install yt-dlp

Poppler auto

PDF text extraction. Installs automatically on first PDF ingest.

brew install poppler

Pandoc auto

Word, Excel, PowerPoint to text. Installs automatically on first use.

brew install pandoc

Web Clipper optional

Browser extension to save articles as markdown directly into your vault.

Install extension →
03

Getting Started

Run the installer

./install.sh

Checks your environment and installs anything missing. Interactive—asks before each tool.

Open Claude Code

claude

Launch in the llm-wiki directory. It reads CLAUDE.md and discovers all available skills.

Create your first vault

/vault-create my-research --domain ai-research

Scaffolds a vault at vaults/my-research/ with full wiki structure and git init.

Open in Obsidian

Open Obsidian → Open folder as vault → select vaults/my-research.

Ingest your first source

/ingest https://some-article.com --vault my-research

Claude reads it, creates wiki pages, updates the index, and auto-commits. Watch pages appear in Obsidian in real time.

Ask questions

/query "What are the key takeaways?" --vault my-research

Synthesized answers with citations. Add --save to file the answer back into the wiki.

Commands

CommandDescription
/vault-create <name>Create a new vault with full wiki structure
/vault-import <path>Import an existing Obsidian vault or markdown folder
/vault-statusShow all vaults with page counts and status
/ingest <source>Process any source into wiki pages
/query "question"Ask questions, get answers with citations
/search "terms"Full-text search across wiki pages
/lintHealth-check wiki for issues
/promote <vault>Graduate knowledge to another vault
/slides "topic"Generate Marp presentation from wiki content
/setupFirst-time machine setup
