Setup Guide

Getting Started

Three commands to install. Six steps to your first wiki. Everything else is optional.

Install

Three commands

The install script checks your environment and offers to install anything missing via Homebrew.

$ git clone https://github.com/RonanCodes/llm-wiki.git
$ cd llm-wiki
$ ./install.sh  # interactive — asks before each tool
Dependencies

What you need

Core tools are required. Everything else installs automatically the first time you need it.

Claude Code required

The AI engine. Reads sources, writes wiki pages, answers questions.

npm i -g @anthropic-ai/claude-code

Docs →

Obsidian required

Your wiki viewer. Graph view, backlinks, search.

brew install --cask obsidian

Download →

Git required

Each vault is its own git repo with full history.

brew install git

Node.js required

Runtime for Claude Code.

brew install node

YouTube transcripts. Installs on first YouTube ingest.

PDF text extraction. Installs on first PDF ingest.

Office doc conversion. Installs on first use.

brew install pandoc

First Vault

Six steps to knowledge

Run the installer

./install.sh

Checks your environment, installs missing tools. Interactive — asks before each one.

Open Claude Code

claude

Launch in the llm-wiki directory. It reads CLAUDE.md and discovers all skills.

Create your first vault

/vault-create my-research --domain ai-research

Scaffolds a vault with full wiki structure and git init. Or run /vault-create with no args for an interactive interview.

Open in Obsidian

Open Obsidian → Open folder as vault → select vaults/my-research. You'll see the wiki structure.

Ingest your first source

/ingest https://some-article.com --vault my-research

Claude reads it, creates wiki pages, updates the index, and auto-commits. Watch pages appear in Obsidian in real time.

Ask questions

/query "What are the key takeaways?" --vault my-research

Synthesized answers with citations. Add --save to file the answer back into the wiki.
