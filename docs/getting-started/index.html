Setup Guide

Getting Started

Three commands to install. Six steps to your first wiki.

Install
$ git clone https://github.com/RonanCodes/llm-wiki.git
$ cd llm-wiki
$ ./install.sh
First Vault

Run the installer

./install.sh

Checks environment, installs missing tools via Homebrew.

Open Claude Code

claude

Launch in the llm-wiki directory.

Create your first vault

/vault-create my-research --domain ai-research

Open in Obsidian

Open Obsidian → Open folder as vault → vaults/my-research

Ingest your first source

/ingest https://some-article.com --vault my-research

Ask questions

/query "What are the key takeaways?" --vault my-research --save
